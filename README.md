# Updated-Online-Book-Library-
An ASP.NET Core MVC digital library system featuring role-based access, SQL Server integration, and paginated book management for seamless borrowing and inventory tracking. 
Developed a web application to manage a digital library system for adding, searching, borrowing, and returning books. The application utilizes ASP.NET Core MVC for the backend, with Entity Framework Core for data access and SQL Server for database management. Integrated features for book management, including role-based user access and pagination for large lists of books.

Roles and Responsibilities:

· Implemented back-end logic for managing books, including APIs for adding, searching, borrowing, and returning books in ASP.NET Core.

· Developed front-end user interface for book management, including search filters, book details, and borrow/return functionalities.

· Designed and managed the SQL Server database schema for storing books, users, borrowing records, and ensuring data integrity.

· Implemented role-based authentication and authorization to allow different user roles (admin and user) for secure access and operations.

· Integrated pagination for efficient handling of large lists of books in the front-end UI.


Tech Stack Used-

· Backend: ASP.NET Core MVC

· Database: Microsoft SQL Server

· ORM: Entity Framework Core

· Frontend: JavaScript
